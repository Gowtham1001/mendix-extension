Browser Test Mode

This tests the OpenRouter API directly from your browser (no Mendix needed).

1. Enter your OpenRouter API key
2. Pick a model (default: cohere/north-mini-code:free)
3. Click Test to verify the connection
4. Start chatting to test streaming responses

Note: CORS is enabled on OpenRouter, so browser requests work.
